rdfa2rdf
========

Standalone application for RDF triple generation from RDFa format.

This program distill (X)HTML+RDFa from :
 - URL
 - local file
 - UI Textarea 

and produces RDF triples in :
 - Turtle
 - N-Triple
 - RDF/XML
 - Json

Launch program
--------------
1. Build it with maven 
2. Double-click on rdfa2rdf.jar (on target folder)
